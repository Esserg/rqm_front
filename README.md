# Random Quote Machine (Frontend)

Это frontend-часть приложения для генерации случайных фраз. Проект написан на React (с использованием TypeScript) и собирается с помощью Vite.

## Требования
- Node.js (версия 18 или выше)
- npm (или yarn/pnpm)
- Docker (опционально, для запуска в контейнере)

## Установка зависимостей
Перед началом работы необходимо установить все зависимости проекта:
```bash
npm install
```

## Сборка и запуск (Локально)

### Режим разработки (Development)
Для запуска приложения в режиме разработчика с горячей перезагрузкой (HMR) выполните:
```bash
npm run dev
```
Обычно Vite запускает сервер по адресу `http://localhost:5173`.

### Сборка для Production
Для создания оптимизированной сборки проекта выполните:
```bash
npm run build
```
Готовые статические файлы будут помещены в директорию `dist`.

### Предпросмотр Production-сборки
Для локального тестирования уже собранного проекта можно использовать встроенный сервер предпросмотра Vite:
```bash
npm run preview
```

## Запуск через Docker
Проект содержит `Dockerfile` и `nginx.conf`, что позволяет легко запустить его в изолированном контейнере с веб-сервером Nginx.

### Сборка Docker-образа
В корневой папке проекта (где находится `Dockerfile`) выполните:
```bash
docker build -t rqm-front .
```

### Запуск Docker-контейнера
Запустите собранный образ, пробросив порт 80:
```bash
docker run --name rqm-frontend -p 80:80 -d rqm-front
```
После этого приложение будет доступно по адресу `http://localhost`.

## Структура конфигурации
- `vite.config.ts` — конфигурация сборщика Vite.
- `tsconfig.json` — настройки TypeScript.
- `nginx.conf` — конфигурационный файл для веб-сервера Nginx (используется внутри Docker).
